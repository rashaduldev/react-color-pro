# Contributing to react-color-pro

Thank you for your interest in contributing to react-color-pro! 🎉

## How to contribute

1. Fork the repository and clone your fork.
2. Create a new branch: `git checkout -b feature/my-new-feature`
3. Make your changes, following the project's code style.
4. Test your changes locally.
5. Submit a pull request describing your changes.

## Code style

- Use TypeScript and React functional components.
- Follow existing formatting and linting rules.
- Write clear, concise commit messages.

## Reporting issues

Please open an issue on GitHub if you find any bugs or have feature requests.
